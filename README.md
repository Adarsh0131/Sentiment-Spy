# Sentiment-Spy

What is Sentiment Spy
The program, Sentiment Spy, is an interactive chatbot powered by TextBlob, designed to analyze the sentiment of user-provided text. The chatbot categorizes messages as positive, negative, or neutral based on their sentiment score and provides real-time feedback with a spy-themed narrative. Users can interact with the chatbot, view summaries of analyzed sentiments, reset the conversation, and receive personalized suggestions based on their mood.
